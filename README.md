# SOC Incident Analysis – Suspicious Login Investigation

## 📖 Overview
This project simulates a cybersecurity incident where multiple failed login attempts were detected, followed by a successful login from an unusual location.

---

## 🧪 Scenario Description
- Multiple failed login attempts detected
- Successful login after failures
- Login from unusual IP/location

---

## 🔍 Analysis

### Observed Events
- High number of failed logins
- Successful login after attempts
- Suspicious IP address

### Possible Attack
- Brute Force Attack
- Credential Stuffing

### Risk Level
High 🚨

---

## 🛡️ Recommendations
- Enable MFA
- Account lockout policy
- Monitor login activity
- Use SIEM tools

---

## 📎 Indicators of Compromise
- Multiple failed logins
- Foreign IP
- Unusual login time

---

## 📊 Conclusion
This incident indicates a possible account compromise and requires immediate security actions.
